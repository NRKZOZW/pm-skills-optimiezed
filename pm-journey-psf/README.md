# pm-journey-psf

Problem Solution Fit (PSF) journey plugin: define and validate solutions for customer problems. Guides PMs through value proposition design, ideation, prototyping, and solution acceptance testing.

## Skills (3)

- **solution-validation** — Score and validate whether a proposed solution truly solves the customer's problem using a structured Solution Validation Scorecard.
- **prototype-test-plan** — Plan and execute a prototype test — from selecting the right fidelity level through writing a test script to synthesizing results.
- **value-proposition-canvas** — Design and evaluate value propositions using the Value Proposition Canvas (Strategyzer) to achieve fit between customer needs and your offering.

## Commands (2)

- `/pm-journey-psf:run-psf` — Run a full Problem Solution Fit cycle — from value proposition design through solution ideation and prototype validation.
- `/pm-journey-psf:design-experiment` — Design a validation experiment for a solution hypothesis — with success criteria, timeline, and decision framework.

## Fit Journey Context

This is **Stage 2** of the Fit Journey (CPF → PSF → SPF → PMF → GTM).

**Inputs**: Validated problem statement, personas, customer insights (from CPF)
**Outputs**: Validated solution concept, value proposition, prototype test results, remaining hypothesis list
**Previous Stage**: Run `/run-cpf` to validate your problem first (pm-journey-cpf)
**Next Stage**: Run `/run-spf` to build and validate your MVP (pm-journey-spf)
